# Hands-On Exercise No. 8 - PHP

This project is a simple PHP MVC-style starter that matches the outputs shown in the assignment screenshot.

## Included outputs

- Output #1: student information form with validation and styled formatted output
- Output #2: home, register, login, and forgot password pages
- Outputs #3 and #4: database-ready student registration form and registered persons list
- Output #5: faculty information module using MVC structure and a primary key in the database

## Run locally

1. Copy the project into your `htdocs` folder if you are using XAMPP.
2. Import `database/hands_on_exercise_8.sql` in phpMyAdmin.
3. Update credentials in `app/core/Database.php` if your MySQL setup is different.
4. Open `http://localhost/HANDS-ON%20EXERCISE%20NO.%208%20-%20PHP/index.php`.
